# UAS-Machine-Learning Heart Disease Prediction using Decision Tree
Proyek ini mengimplementasikan klasifikasi Pohon Keputusan untuk memprediksi penyakit jantung berdasarkan fitur klinis dari dataset UCI Heart Disease.
## Dataset
- Source: UCI Machine Learning Repository
- Samples: 303 patients
- Features: 13 clinical attributes
- Target: Presence of heart disease (binary)
